# CapDeck Releases

This public repository hosts signed update metadata and binary releases for
[CapDeck](https://github.com/jesdx/CapDeck).

`appcast.xml` is signed by the CapDeck Sparkle EdDSA key. Release archives are
published as GitHub Release assets and verified by CapDeck before extraction.

Source code and product documentation remain in the main CapDeck repository.
